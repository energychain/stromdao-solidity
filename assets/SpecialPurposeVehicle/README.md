# Special Purpose Vehilce - Zweckgesellschaft

## Motivation
Die Einnahmen aus einer Erzeugungsanlage sollen entsprechend der Eigentumsanteile verteilt werden. Dies ist besonders bei Bürgergenossenschaften oder Multi-Investoren-Anlagen sinnvoll. Zusätzlich soll die Möglichkeit geschaffen werden, die Eigentumsanteile an einen Dritten weiterzugeben

## Umsetzung
Die Verteilung der Eigentumsanteile werden als Token geführt. Entsprechend den Besitzern dieser Tokens werden Einnahmen, die durch ein Orakel erfasst werden, auf die Stromkonten verteilt.
