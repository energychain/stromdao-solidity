# Stromliefervertrag

Abschluss eines Stromliefervertrages nach einem Angebot-> Annahme Dialoges

Produktiv-Version: [0x5Cd07FCC52db518067933A435d674db917fc7196](https://etherscan.io/address/0x5Cd07FCC52db518067933A435d674db917fc7196)

## Motivation
Der Abschluss eines Stromliefervertrages beruht aus dem Dialog, dass der Interessent seine Postleitzahl mitteilt, daraufhin der Anbieter ein Angebot unterbreitet, welches dann vom Interessenten angenommen wird. 

## Umsetzung
Beim Anlegen eines Angebotes wird ein Contract mitgegeben, welcher den eigentlichen Vertrag beinhaltet. Die "Negotiation"  (Verhandlung) dient zur Absicherung des Abschlussprozesses nach dem Prozess Angebot -> Annahme -> Abschluss. 

## Blockchain
Die Nutzung der Blockchain erlaubt es mit einer höheren Frequenz die Verträge zu wechseln und dabei die Wechselprozesse zu automatisieren. Die im Source Code enthaltenen Beispiele können als Ergänzung zum Stromkonto verwendet werden.
